Functions:
Модуль з багаторазовими функціями, використовується тільки іншими модулями бота. 

save_to_txt - Підфункція отримує на вході словник з введеними данними та зберігає дані до файлу, назва файлу залежить від id користувача, 
	файл зберігається в папку data, підфункція, повертає головній функції True якщо все вдалося.

check_user - Підфункція перевірки чи є анкета користувача, викликається при спробі створити нову анкету, 
	повертає True якщо існує анкета та є запис в файлі users, повертає False, якщо запису та файлу немає.

see_file - Підфункція викликає функцію check_user, якщо повертається значення True  функція зчитує файл анкети, повертає строку.

delete_file - Підфункція викликає функцію check_user, якщо повертається значення True функція видаляє файл анкети та редагує файл users, повертає рядок.


Keyboards:
Модуль з функціями, які формують клавіатури.

fao_keyboard - Підфункція формування клавіатури з FAO бота, повертає масив масивів об`єкту InlineKeyboardButton.

work_keyboard - Підфункція формування клавіатури з пропозиціями про роботу, повертає масив об`єкту InlineKeyboardButton.

menu_keyboard - Підфункція формування клавіатури з функціями(командами) боту, повертає масив масивів об`єкту KeyboardButton.

file_menu_keyboard - Підфункція формування клавіатури з функціями(командами) боту, повертає масив масивів об`єкту KeyboardButton

reg_keyboard - Підфункція формування клавіатури з функціями, боту повертає масив об`єкту KeyboardButton


Callbacks:
Зворотні виклики які реагують на натискання клавіатур.
fao_callback_button - зворотні виклики на натиск кнопок клавіатури створеної за допомогою підфункції fao_keyboard,
	повертає повідомлення з текстом користувачу.


work_callback_button - зворотні виклики на натиск кнопок клавіатури створеної за допомогою підфункції work_keyboard,
	повертає повідомлення з текстом користувачу.


Commands:
Модуль з командами бота, які може вводити користувач.

/start - функція програми викликає підфункцію menu_keyboard, повертає користувачу текстове повідомлення та інлайн клавіатуру.

Меню - функція програми викликає підфункцію menu_keyboard, повертає користувачу текстове повідомлення та інлайн клавіатуру,
	якщо включений кінцевий автомат, то виходить з нього.

Допомога - функція програми повертає користувачу текстове повідомлення з описом як шукати роботу та загальними відомостями.

ФАО - функція програми викликає підфункцію fao_keyboard повертає користувачу текстове повідомлення та інлайн клавіатуру.

Гід - функція програми повертає користувачу текстове повідомлення.

Відіслати резюме - функція програми повертає користувачу текстове повідомлення з запросом документів, 
	після отримання документу бот зберігає документ з назвою в виді id користувача в папці downloads/documents.

Робота - функція програми викликає підфункцію work, повертає користувачу текстове повідомлення з інлайн клавіатурою з назвами вакансій.

Переглянути анкету - функція програми викликає підфункцію see_file, повертає користувачу текстове повідомлення.

Видалити анкету - функція програми викликає підфункцію delete_file, повертає користувачу текстове повідомлення.

Реєстрація - функція кінцевого автомату, повертає користувачеві текстове повідомлення, клавіатуру підфункцією cancel_keyboard,
починає послідовний прийом повідомлень.

Скачувати реєстрацію - підфункція кінцевого автомату повертає користувачеві текстове повідомлення, клавіатуру підфункцією  menu_keyboard,
закінчує послідовний прийом повідомлень.

Відміна - підфункція кінцевого автомату повертає користувачеві текстове повідомлення, клавіатуру підфункцією  reg_keyboard,
повертає кінцевий автомат до нейтрального стану.


echo - функція програми, отримує на вхід всі повідомлення якщо кінцевий автомат не включений, 
	або команда бота не існує,повертає користувачу повідомлення:"Не зрозумів вашої команди. Перевірте правильність вводу".
